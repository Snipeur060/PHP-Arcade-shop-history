Endpoint: /api/productinfo

Méthode HTTP: GET

Cet endpoint retourne les détails d'un produit spécifique en utilisant son nom.

Paramètres

  • productname: (Requis) Le nom du produit dont vous souhaitez obtenir les détails.

Réponse

[
  {
    "id": "212",
    "productname": "Octocat",
    "hours": "15",
    "imageURL": "https://cloud-n716uf1pe-hack-club-bot.vercel.app/0gh_0008_z_dd7b_copy.png",
    "date": "2024-08-29 23:13:07",
    "stock": "0"
  }
]

Exemple de requête

https://arcade-tracker.snipeur060.fr/api/productinfo?productname=Octocat

Cette endpoint permet d'avoir des infos sur un produit ayant une id special

Paramètres
